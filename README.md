## 環境

- next
- react
- tailwind css
- nextui
- bun

## 環境構築

- node がインストールされているか確認

```bash
$ node -v
v18.17.0
```

Node のバージョン管理は各々で選択して 👌

- bun がインストールされているか確認

```bash
$ bun -v
1.0.2
```

インストールされていない場合は、以下を実行する

```bash
$ brew install bun
```

- 依存関係をインストール

```bash
$ bun install
```

- 起動

```bash
$ bun run dev
```
